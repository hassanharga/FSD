# Project Documentation

## Running the Project

### Prerequisites

1. Install Docker

2. Start MongoDB using Docker Compose:
   ```sh
   docker-compose up -d
   ```

### Frontend

1. Navigate to the frontend directory:
   ```sh
   cd frontend
   ```
2. Install dependencies:
   ```sh
   pnpm install
   ```
3. Start the development server:
   ```sh
   pnpm dev
   ```

### Backend

1. Navigate to the backend directory:
   ```sh
   cd backend
   ```
2. Install dependencies:
   ```sh
   pnpm install
   ```
3. Start the server:
   ```sh
   pnpm start:dev
   ```
